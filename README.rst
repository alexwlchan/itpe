ITPE
====

Scripts for generating the masterpost HTML for ITPE (the Informal Twitter
Podfic Exchange).

Installation
------------

Install these scripts by running:

.. code-block:: console

   pip install itpe

or:

.. code-block:: console

   pipsi install itpe

Usage
-----

Varies from year to year.  A good starting point is:

.. code-block:: console

   itpe --help

after you've installed the scripts.
